# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제:

안전 화학 반응 설계 및 대체 물질 추천 시스템

---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.


내 설명:사용자가 실험에 사용하고 싶은 여러 가지 화학 물질들을 입력하면, 프로그램이 위험 등급 순으로 정렬하여 고위험 물질을 감지한다. 그 후 안전한 대체 물질을 다지선다형으로 추천하여 사용자가 선택하게 한다. 그리고 사용자가 선택한 물질을 실험 재료에 포함시켜 프로그램이 추천했던 실험을 진행한다. 그때  실험실 환경(온도, 압력) 조건에 따른 안전성 및 실험 오차 가능성을 시뮬레이션 해 최적의 결과를 추천한다.


---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- 실험에 사용하고 싶은 여러 화학 물질들
- 실험실 측정 온도
- 실험실 측정 압력
- 프로그램 추천 대체 물질 중 사용자가 택한 물질의 번호

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 사용자가 입력한 물질들 리스트로 정리
- 위험도 기준 내림차순 물질 정렬
- 고위험 물질 판별 및 수정할지 결정
- 실험 추천 및 환경 조건 시뮬레이션

### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 초기에 입력한 물질들 위험한 정도를 기준으로 내림차순으로 정렬
- 위험 진단 및 대체 물질 추천
- 최종 재료 라인업 및 실험 추천
- 환경 시뮬레이션 결과 및 오차 진단

## 6. 사용할 제어구조
1. 순차 구조 (Sequence)
적용 부분: main() 함수 전체 흐름

설명: 사용자가 재료와 환경 데이터를 입력하면 ➡️ 사전 매칭 ➡️ 위험도 정렬 ➡️ 고위험 물질 대체 ➡️ 실험 추천 및 시뮬레이션 출력까지 위에서 아래로 순서대로 실행됨.

2. 반복 구조 (Loop)
적용 부분: for 문 (get_chemical_info 및 replace_danger_material 함수)

설명: 사용자가 입력한 여러 개의 물질 리스트를 for 문으로 하나씩 꺼내며 2차원 리스트인 화학 물질 사전을 한 줄씩 처음부터 끝까지 탐색(선형 탐색)할 때 사용함.

3. 선택 구조 (Condition)
적용 부분: if-else 문 (replace_danger_material 및 recommend_and_simulate 함수)

설명: [프로그램의 핵심 분기점]

추출된 물질 중 위험 등급 == 3인지 판별하여 고위험 물질일 때만 대체재 추천창을 띄우는 조건문.

실험실의 현재 온도가 최적 온도 범위 내에 있는지, 아니면 오차가 크게 발생했는지 비교 연산자(<, >, ==)를 통해 판정하는 조건문.

### 조건문을 사용할 부분

예) 사용자의 예산보다 가격이 낮은 항목만 추천한다.

내 계획:
고위험 물질 감지 및 예외 처리 (replace_danger_material 함수)

추출된 물질 데이터를 검사하다가 위험 등급 == 3인 경우를 판별함.

조건 만족 시 (True): 화면에 빨간색 경고를 띄우고 다지선다형 대체재 선택 메뉴를 실행함.

조건 미만족 시 (False): 안전한 물질이므로 교체 프로세스를 건너뛰고 그대로 통과시킴.

실험실 환경 및 오차 판정 (recommend_and_simulate 함수)

사용자가 입력한 현재 온도/압력을 experiment_db에 저장된 최적 조건과 비교 연산자(<, >, ==)로 비교함.

현재 온도가 한계 온도를 초과했는지 조건문으로 감지하여 "실험 불가(위험)" 또는 "오차 발생(주의)" 등의 안전 메시지를 분기 출력함.


### 반복문을 사용할 부분

예) 2차원 리스트의 모든 항목을 하나씩 검사한다.

내 계획:
입력 데이터의 선형 탐색 (get_chemical_info 함수)

사용자가 입력한 4~6개의 물질 목록(user_list)에서 이름을 하나씩 순서대로 꺼내기 위해 for 문을 사용함.

꺼낸 물질 이름이 데이터베이스인 chemical_db(2차원 리스트)의 몇 번째 행에 존재하는지 위에서부터 한 줄씩 끝까지 비교·탐색할 때 중첩 반복문을 활용함.

위험 물질 전수 검사 (replace_danger_material 함수)

선택된 전체 물질 리스트를 처음부터 끝까지 for 문으로 순회하며, 고위험 물질(3등급)이 포함되어 있는지 누락 없이 전수 검사할 때 활용함.
